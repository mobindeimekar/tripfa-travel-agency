# Tripfa | تریپفا

وب‌سایت معرفی خدمات هواپیمایی و گردشگری تریپفا با تمرکز بر تجربه کاربری فارسی، طراحی مینیمال و سکشن‌های کاربردی برای رزرو، مقاصد محبوب و معرفی خدمات.

## توضیح پروژه
این پروژه یک لندینگ‌پیج تک‌صفحه‌ای برای یک آژانس مسافرتی است که شامل هدر ثابت، هرو با ویدیوی پس‌زمینه، فرم جستجوی سفر با تب‌های مختلف، مقاصد منتخب با داده‌های پویا، درباره ما، معرفی اپلیکیشن و بخش پرسش‌های متداول است.

## ویژگی‌ها
- طراحی راست‌به‌چپ (RTL) و مناسب زبان فارسی
- هدر چسبان و منوی کشویی موبایل
- هرو با ویدیوی پس‌زمینه و CTAهای واضح
- فرم جستجوی سفر با تب‌ها و حالت رفت‌وبرگشت
- کارت‌های مقصد با داده‌های پویا (JS)
- بخش FAQ با آکاردئون تعاملی
- ریسپانسیو برای موبایل و دسکتاپ

## تکنولوژی‌ها
- HTML5
- CSS3 (Flexbox, Grid, Variables)
- JavaScript (Vanilla)
- Font Awesome

## توضیح معماری
- ساختار تک‌صفحه‌ای در `index.html` که تمام سکشن‌ها را نگه می‌دارد.
- استایل‌ها در `assets/css/style.css` شامل توکن‌های رنگی، تایپوگرافی و لایه‌بندی ریسپانسیو هستند.
- منطق تعاملی در `assets/js/main.js` (تب‌ها، دراور موبایل، آکاردئون FAQ، و رندر مقاصد).
- داده‌های مقاصد در `assets/js/data.js` نگهداری می‌شوند تا کارت‌ها به‌صورت پویا ساخته شوند.
- فایل‌های رسانه‌ای در `assets/media/` و فونت‌ها در `fonts/` ذخیره شده‌اند.

## ساختار پوشه‌ها
```text
project 14 tripfa/
  assets/
    css/
      style.css
    js/
      data.js
      main.js
    media/
      hero.mp4
      ...
  fonts/
  favicon.png
  index.html
  logo.png
```

## اجرای محلی
- کافی است فایل `index.html` را در مرورگر باز کنید.
- یا از Live Server در VS Code استفاده کنید.
